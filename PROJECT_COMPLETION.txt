================================================================================
INSTALLATION & USAGE
================================================================================

QUICK START:
  1. cd kalshi-sentiment-analyzer
  2. ./setup.sh
  3. source venv/bin/activate
  4. python run_analysis.py

ALTERNATIVE:
  1. pip install -r requirements.txt
  2. python run_analysis.py

JUPYTER:
  1. pip install -r requirements.txt
  2. jupyter notebook notebooks/analysis.ipynb
